# Text Reverser by Word

A Python script that reverses the order of words in a sentence.

## Description
This project takes a sentence as input, splits it into words, reverses the order, and joins them back into a sentence.

## Features
- Word order reversal
- Simple string manipulation
- Beginner-friendly logic

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python word_reverser.py

4. Enter a sentence when prompted

## Example

Input:
I love Python programming

Output:
programming Python love I
